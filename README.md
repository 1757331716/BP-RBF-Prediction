# BP-RBF-Prediction
使用BP神经网络、RBF神经网络以及PSO优化的RBF神经网络进行数据的预测。  
数据采用的是（52，4）的shape，分为了训练集和测试集，对四个特征（由num=1,2,3,4参数控制）分别进行预测。  
最后会输出神经网络分别在训练集和测试集上的效果图，以及在训练集和测试集上的误差。
